# flashcard-pwa

A personnal project to learn more about JavaScript and Progressive Web Apps (PWAs).

Check out the [Live Demo](https://gaspardlbrg.github.io/flashcard-pwa/).

---

## Roadmap

### Phase I · Beginnings
* [ ] Set up the homepage
* [ ] Making the logic for basic flashcards

### Phase II · Upbringing
* [ ] Flashcards compatible with TeX equations (Mathjax probably)
* [ ] Flashcards compatible with Images
* [ ] Flashcards saved as JSON files

### Phase III · True PWA
* [ ] A menu where you can access all of your flashcards
* [ ] Easy storing, modifying, and creating of flashcard

---

## About the project

This application is built entirely from scratch as a learning exercise to better understand web development, service workers, and local storage.

Made by [Gaspard Le Borgne]{https://gaspard-lb.web.app/}. 100% AI-free code.
